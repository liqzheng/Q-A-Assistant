## 📋 Detailed Version (Optional)

**AI PDF Q&A Assistant**

Developed a full-stack AI-powered document Q&A system that allows users to upload PDF documents and ask questions in natural language, receiving accurate answers based on document content.

**Technical Stack:**
- Frontend: React 18, Ant Design 5, Axios
- Backend: Node.js, Express.js, Multer
- AI/ML: OpenAI API (text-embedding-3-small, GPT-3.5-turbo)
- Core Technologies: Vector Embeddings, Cosine Similarity, RAG

**Key Achievements:**
- Implemented intelligent text chunking algorithm with overlap for context preservation
- Built in-memory vector database with cosine similarity search
- Integrated RAG (Retrieval-Augmented Generation) pipeline
- Designed RESTful API architecture
- Created responsive, modern UI with Ant Design

**Technologies:** React, Node.js, Express, OpenAI API, Vector Search, RAG, Semantic Search, PDF Processing
