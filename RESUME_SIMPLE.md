# 简历和 LinkedIn 项目描述（简洁版）

## 📝 简历项目描述

**AI PDF 智能问答助手**

基于向量检索和 GPT 的智能文档问答系统。实现 PDF 文档上传、自动解析、向量化存储和语义检索功能，使用 React + Node.js 构建全栈应用。

**技术栈：** React, Node.js, Express, OpenAI API, 向量检索, RAG, Ant Design

**核心功能：**
- PDF 文档自动解析和文本切分
- 基于向量相似度的语义检索
- 集成 GPT-3.5 实现智能问答
- React + Ant Design 现代化 UI

## 🔗 LinkedIn 项目描述

**AI PDF 智能问答助手 | 全栈开发**

开发了一个基于 AI 的 PDF 文档智能问答系统，实现文档上传、向量化存储和智能问答功能。

**技术实现：**
- 前端：React + Ant Design

**项目亮点：**
✨ 实现 PDF 文档的自动解析和智能切分
✨ 使用向量嵌入实现语义检索
✨ 集成 GPT-3.5 实现基于文档的问答
