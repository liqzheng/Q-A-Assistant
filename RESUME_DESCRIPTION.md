# 简历和 LinkedIn 项目描述

## 📋 项目名称
**AI PDF 智能问答助手** (AI PDF Q&A Assistant)

---

## 🎯 项目简介（简短版 - 适合简历）

基于向量检索和 GPT 的智能文档问答系统，支持 PDF 文档上传、自动解析、向量化存储和智能问答。使用 React + Node.js 构建全栈应用，实现文档内容的语义检索和 AI 驱动的问答功能。

---

## 📝 详细描述（适合 LinkedIn 和简历详细版）

### 项目概述
开发了一个基于 AI 的 PDF 文档智能问答系统，用户可以通过自然语言提问，系统基于文档内容提供精准回答。项目采用前后端分离架构，实现了从文档上传、文本解析、向量化存储到智能问答的完整流程。

### 核心功能
- **PDF 文档处理**：支持 PDF 文件上传，自动提取文本内容并智能切分
- **向量化存储**：使用 OpenAI Embeddings API 将文档内容转换为向量嵌入，实现语义检索
- **智能问答**：基于向量相似度检索相关文档片段，结合 GPT-3.5 生成精准回答
- **现代化 UI**：使用 Ant Design 构建响应式用户界面，提供流畅的交互体验

### 技术实现亮点
- **文本切分策略**：实现智能文本分块算法，支持重叠切分，保证上下文完整性
- **向量检索**：使用余弦相似度算法进行语义检索，提高问答准确性
- **内存向量存储**：实现高效的内存向量数据库，支持快速检索（可扩展为 Chroma/FAISS）
- **RESTful API 设计**：设计清晰的 API 接口，支持文档管理和问答功能

## 🛠️ 技术栈

### 前端
- **React 18** - 用户界面框架
- **Ant Design 5** - UI 组件库
- **Axios** - HTTP 客户端
- **React Hooks** - 状态管理

### 后端
- **Node.js** - 运行时环境
- **Express.js** - Web 框架
- **Multer** - 文件上传处理
- **pdf-parse** - PDF 文本提取
- **OpenAI API** - 向量嵌入和文本生成
- **内存向量存储** - 向量数据库（可扩展为 Chroma/FAISS）

### 核心技术
- **向量嵌入 (Embeddings)** - 使用 OpenAI text-embedding-3-small 模型
- **语义检索** - 余弦相似度算法
- **RAG (Retrieval-Augmented Generation)** - 检索增强生成
- **文本处理** - 智能文本切分和预处理
